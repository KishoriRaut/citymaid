# Public Assets

## QR Code Setup

To add your payment QR code:

1. Save your QR code image as `qr-code.png`
2. Place it in this `public` folder
3. The image will be accessible at `/qr-code.png`

### Recommended QR Code Specifications:
- Format: PNG or JPG
- Size: 512x512 pixels or larger (square)
- Content: Should contain payment information for Rs. 3,000

The QR code will automatically display on the unlock contact page when users select "QR Code" as the payment method.
